Job Title: Senior Software Engineer (Backend)

Company: TechScale Inc
Location: Remote / On-site
Type: Full-Time

About the Role:
We are hiring a Senior Software Engineer to build and maintain scalable backend
services that power our platform. You will design APIs, optimize databases, and
ensure high availability across our microservices architecture.

Required Skills:
- Programming: Python, Java, or Go (at least 2)
- Web Frameworks: Django, Flask, FastAPI, or Spring Boot
- Databases: PostgreSQL, MySQL, MongoDB, Redis
- API Design: RESTful APIs, GraphQL
- Cloud Platforms: AWS (EC2, Lambda, RDS, S3) or GCP
- Containerization: Docker, Kubernetes
- CI/CD: GitHub Actions, Jenkins, or GitLab CI
- Version Control: Git, GitHub

Preferred / Bonus Skills:
- Message Queues: Kafka, RabbitMQ, or SQS
- Infrastructure as Code: Terraform or CloudFormation
- Monitoring: Prometheus, Grafana, DataDog
- Microservices architecture patterns
- gRPC and Protocol Buffers
- System design experience

Experience Requirements:
- 4+ years of backend engineering experience
- Experience with systems handling 10K+ requests/second
- Track record of shipping production features end-to-end

Responsibilities:
- Design and build RESTful APIs and microservices
- Optimize database queries and system performance
- Write comprehensive unit and integration tests
- Participate in code reviews and architecture discussions
- On-call rotation for production systems
- Mentor junior engineers
